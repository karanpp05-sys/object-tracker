How It Works

Secure Smart Recovery

A two-step verification system prevents false claims and ensures items always reach their rightful owners.

Report Lost or Found

Quickly submit items with photos, detailed description, color, brand, location, and unique features.

Dynamic Quiz Verification

4 questions dynamically generated from item details. Only the real owner answers correctly. Requires 60%+ to pass.

OpenCV Image Matching

Advanced ORB feature matching compares uploaded photo with found item image. Requires 60%+ similarity to pass.

Admin Review & Approval

Admin sees both scores and detailed results before approving or rejecting each claim request.
